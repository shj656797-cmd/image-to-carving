# 图片转雕刻图网站

一个功能完整的图片转雕刻图网站，支持多种雕刻效果处理。

部署在 GitHub Pages 上，完全静态网站，无需后端服务器。

## 功能特性

- 🖼️ **多种雕刻效果**
  - 灰度图转换
  - 浮雕效果
  - 木雕纹理
  - 边缘检测

- 📤 **图片上传处理**
  - 支持本地图片上传
  - 实时预览效果
  - 下载处理后的图片

- 🎨 **参数调整**
  - 可调整处理强度
  - 实时效果预览
  - 多种预设方案

- ⚡ **完全客户端处理**
  - 无需后端服务器
  - 快速高效的图像处理
  - 隐私保护（图片不离开浏览器）

## 快速开始

### 在线使用

访问 GitHub Pages 部署的网站：
```
https://shj656797-cmd.github.io/image-to-carving
```

### 本地运行

1. 克隆仓库：
```bash
git clone https://github.com/shj656797-cmd/image-to-carving.git
cd image-to-carving
```

2. 启动本地服务器（Python 3）：
```bash
python -m http.server 8000
```

3. 在浏览器中打开：
```
http://localhost:8000
```

## 支持的效果

- **灰度图**：将彩色图像转换为灰度
  - 支持对比度调整
  
- **浮雕**：产生3D浮雕视觉效果
  - 可调整浮雕角度（0-360°）
  - 使用 Sobel 边缘检测算法
  
- **木雕**：模拟木雕纹理效果
  - 支持 3 种纹理方向：横向、纵向、斜向
  - 高斯模糊 + 定向卷积核
  
- **边缘检测**：提取图像边缘
  - 基于 Canny 算法的简化版
  - 清晰的边界提取

## 技术栈

- **前端**：HTML5 + CSS3 + JavaScript
- **图像处理**：Canvas API + Custom Algorithms
- **部署**：GitHub Pages
